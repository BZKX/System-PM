# FINAL_iteration_1

## 项目总结 (Iteration 1)

本次迭代重点解决了用户提出的易用性和可视化维度问题。

### 1. 核心改进
*   **多维视角**: 引入了“计划视图”，让 PM 既能看资源（系统视角），也能看项目进度（计划视角）。
*   **效率工具**: 新增“排期冲突速查”，大大缩短了 PM 回复业务方咨询的时间，无需繁琐的表单录入即可预知风险。
*   **视觉升级**: 系统管理从枯燥的表格转变为现代化的卡片视图，提升了产品的交互质感。

### 2. 下一步建议
*   **数据导入导出**: 支持 Excel 导入计划，方便批量迁移历史数据。
*   **通知提醒**: 当冲突发生时，通过邮件或消息通知相关负责人（需后端支持）。
*   **更细粒度的权限**: 区分 PM（读写）和普通用户（只读）。

### 3. 操作指引
*   访问 Dashboard 体验新的视图切换和速查工具。
*   访问 Systems 管理页体验新的卡片交互。
