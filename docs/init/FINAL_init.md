# FINAL_init

## 1. 项目总结
本项目成功实现了一个基于 Web 的 **系统PM管理台**，解决了多上线计划下的系统资源冲突管理问题。

### 1.1 核心成果
*   **可视化排期仪表盘**: 实现了基于时间轴的系统资源视图，直观展示了各系统的占用情况。
*   **冲突检测引擎**: 实现了实时的冲突检测逻辑，能够精确识别“外灰”和“全网”阶段的资源冲突，并在 UI 上进行高亮预警。
*   **计划与系统管理**: 提供了完整的 CRUD 功能，支持快速录入和管理计划及系统数据。

### 1.2 技术亮点
*   **ConflictEngine**: 独立的冲突检测工具类，逻辑清晰，易于扩展。
*   **Custom Timeline**: 使用 Flexbox + Tailwind 自定义实现的轻量级时间轴组件，比通用图表库更贴合业务需求。
*   **Zustand Store**: 使用 Zustand 进行状态管理并集成了 `persist` 中间件，实现了无后端的本地数据持久化。

## 4. 操作指引
1.  运行 `pnpm install` 安装依赖。
2.  运行 `pnpm run dev` 启动本地服务。
3.  访问 `http://localhost:5173`。
4.  初次访问会自动加载 Mock 数据，可直接体验冲突检测功能。
